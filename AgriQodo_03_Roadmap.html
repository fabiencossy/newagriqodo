Pitch en une phrase

Agri Qodo est l'ERP métier de l'exploitation agricole suisse : la conformité PER au quotidien, la gestion comptable et commerciale via Odoo, et le pilotage économique de la ferme dans une seule application — pensée d'abord pour la Romandie, déployable ensuite en Suisse alémanique et au Tessin.

INTÉGRATION CLÉ

Connexion à digiagrifood.ch / agridata.ch

Agri Qodo se raccordera au data space agroalimentaire suisse (agridata.ch, opérationnel depuis 2023) pour appliquer le principe once-only : l'agriculteur saisit une donnée une seule fois, les autres applications autorisées la consomment via agridata.ch sans ressaisie. Module dédié M14 prévu en V2.

SOCLE TECHNIQUE

Odoo Enterprise obligatoire

Odoo Enterprise (Online ou Odoo.sh) est un prérequis fonctionnel — pas de mode standalone. Intégration incluse sans surcoût dans tous les plans. Modules Odoo custom optionnels réservés aux clients Odoo.sh (segment grandes structures).

LICENCE

100 % open source

Code source publié (recommandation initiale : AGPL v3). Permet l'audit, la contribution communautaire et l'interopérabilité avec d'autres logiciels agricoles suisses. Cohérent avec la philosophie de digiagrifood.ch.

DISTRIBUTION

iOS · Android · Web

Apps natives sur App Store et Play Store + version web responsive (ordinateur et téléphone). Une seule base de code (React Native + Next.js). Souscription sur le web pour éviter la commission Apple/Google.

FONCTIONNEMENT

Hors ligne complet

Architecture offline-first : saisie, calcul du Suisse-Bilanz, consultation, tout fonctionne sans réseau. Synchronisation différentielle au retour en ligne avec détection des doublons et résolution explicite des conflits.

PROMESSE PRODUIT

Puissance + simplicité radicale

Un agriculteur de 60 ans qui n'a jamais ouvert un logiciel doit pouvoir, sans formation, sans tutoriel, sans appel au support : créer ses parcelles, saisir une intervention phyto en moins de 30 secondes, émettre un bon de commande, consulter son Suisse-Bilanz. Si l'une de ces 4 actions exige une assistance, le produit a échoué.

Carte de dépendances entre modules

M1 Parcellaire est la fondation : tous les modules MVP en dépendent (flèches vertes). M3 Suisse-Bilanz est le module le plus « tirant » sur les autres. M10 Comptabilité agricole est le pivot ERP.

Lecture rapide

M1 Parcellaire est le module-fondation : tous les modules MVP en dépendent (flèches vertes). À l'onboarding, c'est le premier module à compléter — il doit être ultra-simple à remplir (idéalement par import GIS cantonal). M3 Suisse-Bilanz agrège M2 (engrais épandus), M4 (animaux pour le calcul UGB), et plus tard M8 (achats d'engrais). C'est le module le plus « tirant » sur les autres.

Pricing volontairement simple — deux plans seulement. Gratuit avec 1 module et 1 utilisateur. Plan unique payant dès qu'on prend 2 modules ou plus, peu importe le nombre. Forfait jusqu'à 10 utilisateurs, surfacturation au-delà. Intégration Odoo incluse, mode hors ligne inclus.

Option Enterprise (modules Odoo custom)

Pour les grandes structures qui ont besoin de modules Odoo personnalisés, un setup additionnel sur Odoo.sh est requis. Tarification : à partir de 2 500 CHF de setup + 50 CHF/mois de maintenance par module custom, en plus du Plan unique. Inclut SLA contractuel, support dédié, onboarding personnalisé et formation sur site.
