Accessibilité

  • Modal : role="dialog" + aria-modal="true" + aria-label.
  • Trigger : aria-haspopup="dialog" + aria-expanded.
  • Résultats : role="listbox" + aria-multiselectable selon mode + items role="option".
  • Catégories : aria-pressed sur l'item actif.
  • Focus trap activé à l'ouverture, focus revient sur le trigger à la fermeture.
  • Esc ferme, Enter sélectionne, ↑/↓ parcourt les résultats, Tab navigue entre input/categories/results.
  • Cibles tactiles : 40 px (résultats), 36 px (catégories), 44 px (trigger).
  • Lecteur d'écran : compteur "X sélectionnés sur Y" annoncé via aria-live="polite".
