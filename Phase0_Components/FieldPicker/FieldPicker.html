Accessibilité

  • Desktop popup : role="dialog" ou role="listbox" selon mode, fermeture Esc + clic extérieur.
  • Mobile fullscreen : role="dialog" + aria-modal="true", focus trap, fermeture via bouton retour.
  • Trigger : aria-haspopup="dialog" + aria-expanded.
  • Catégories : role="tablist" + aria-pressed sur l'actif.
  • Items : role="option" + aria-selected, aria-multiselectable sur la liste en mode multi.
  • Compteur de sélection : aria-live="polite".
  • Bouton OK mobile : disabled si aucune sélection (single).
  • Cibles tactiles : 44 px (items mobile), 36 px (chips catégories), 40 px (items popup desktop).
