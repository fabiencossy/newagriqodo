Accessibilité

  • Canvas carte : role="application" + aria-label.
  • Boutons de contrôle : aria-label explicite (zoom, position, plein écran).
  • Toggle basemap : role="radiogroup" + aria-pressed.
  • Aside : role="complementary".
  • Mobile bottom sheet : focus trap quand ouvert, dismissable swipe-down ou Esc.
  • Navigation clavier prévue : Tab pour controls, +/- pour zoom, flèches pour pan, Enter pour sélection.
  • Cibles tactiles ≥ 36 px (contrôles), 44 px (FAB, boutons sheet).
