Accessibilité

  • role="search" sur le conteneur racine.
  • Facets : chaque bouton de retrait a un aria-label explicite ("Retirer le filtre Culture").
  • Suggestions : role="listbox" + items role="option" + aria-selected.
  • Boutons d'action : aria-expanded sur Filtres / Regrouper / Favoris.
  • Navigation clavier : ↑/↓ pour parcourir suggestions, Enter pour valider, Backspace efface la dernière facet quand input vide.
  • Contraste primary #2d5016 sur fond #fff = 8.9:1 ✓, accent #875a7b sur #fff = 5.5:1 ✓.
  • Cibles tactiles ≥ 32 px (icônes) / 44 px (barre principale).
