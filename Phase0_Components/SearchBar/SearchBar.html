Accessibilité

  • role="search" sur la barre.
  • Boutons icône (loupe, filtre, chevron) : aria-label explicite.
  • Bouton filtre : aria-pressed reflète l'état (au moins un filtre actif → pressed).
  • Bouton chevron : aria-haspopup="menu" + aria-expanded.
  • Facets : bouton retrait avec aria-label explicite ("Retirer Mes devis").
  • Suggestions : role="listbox" + role="option" + aria-selected.
  • Navigation clavier : ↑/↓ suggestions, Enter valide, Backspace efface dernière facet quand input vide, Esc ferme.
  • Contraste vérifié light : texte #1a1a1a sur #fff = 18.6:1 ✓ ; aubergine sur fond clair = 5.5:1 ✓.
  • Cibles tactiles : barre 36 px desktop / 40 px mobile, boutons internes 26 px (zone élargie au padding).
