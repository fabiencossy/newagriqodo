Accessibilité

  • aria-expanded + aria-controls sur le bouton "Filtres avancés"
  • Pills : chaque bouton de retrait a un aria-label spécifique
  • Contraste texte ≥ 4.5:1 (vérifié : #1a1a1a sur #ffffff = 18.6:1)
  • Cibles tactiles ≥ 44 × 44 px (input height 44px, boutons min 32px + padding)
  • Navigation clavier : Tab → input → clear → bouton avancé → champs avancés
