Accessibilité

  • Toggle mode : role="tablist" + aria-pressed.
  • Pauses : role="group" avec label dédié.
  • Calcul auto : aria-live="polite" pour annoncer le total.
  • Validation : aria-invalid sur inputs + message role="alert".
  • Tous les boutons ajout/retrait ont un aria-label.
  • Cibles tactiles 44 px (inputs principaux), 38 px (inputs pauses).
  • Bouton trash sur pause = action destructive → confirmation pas nécessaire (action minime, undo via "Ajouter").
