ViewSwitcher — Wireframe

Toggle entre Table / Carte / Dashboard. Segmented control sur desktop, dropdown sur mobile. Icônes SVG inline 20 px (style Lucide, 1.5 px stroke). Plus d'emoji.

Desktop — Segmented (icône + label)

Vue active : Table

Desktop — Variante icône seule (compact)

Desktop — Hover (tooltip)

Desktop — Disabled

Mobile (< 768px) — Dropdown

Accessibilité

  • Segmented : role="tablist" + boutons role="tab" + aria-pressed.
  • Mobile : aria-haspopup="listbox", options role="option", aria-selected.
  • Variant icône seule : aria-label obligatoire + tooltip au hover.
  • Icônes SVG inline (style Lucide, 1.5 px stroke, currentColor) → cohérent avec le texte.
  • Navigation clavier : ←/→ sur segmented, ↑/↓ sur dropdown, Enter pour sélectionner.
  • Cibles tactiles ≥ 36 px desktop, 44 px mobile.
