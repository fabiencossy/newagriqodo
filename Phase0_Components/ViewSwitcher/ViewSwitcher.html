ViewSwitcher — Wireframe

Toggle entre Table / Carte / Dashboard pour une même donnée. Segmented control sur desktop, dropdown sur mobile. Icônes 20 px. Tooltip délai ~400 ms.

Desktop — Segmented control

Vue active : Table

Desktop — Hover state (tooltip)

Desktop — Disabled

Mobile (< 768px) — Dropdown

Accessibilité

  • Segmented : role="tablist" + boutons role="tab" + aria-pressed.
  • Mobile : aria-haspopup="listbox", options role="option", aria-selected.
  • Toutes les vues ont un libellé textuel + icône (pas d'icône-seul).
  • Navigation clavier : ←/→ sur segmented, ↑/↓ sur dropdown, Enter pour sélectionner.
  • Cibles tactiles ≥ 44 px (h:36px desktop / h:44px mobile + zone tap).
